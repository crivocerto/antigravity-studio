# Design: Hydration Transparente

Não haverá alterações no layout ou UI, visto que a responsabilidade desta spec é apenas "Data Hydration" (Hidratação de dados).

O objetivo de design aqui é que **o usuário final jamais consiga perceber a diferença** entre um post inserido manualmente com colunas perfeitas, e um post inserido por uma IA onde tudo estava espremido dentro de um único JSON. O componente de `PostCard` renderizará ambos com a exata mesma fidelidade do "Apple Liquid Glass" construído até aqui.
